# AI Interviewer 

AI Interviewer is AI-powered mock interview platform that helps users prepare for real interviews through intelligent question generation, voice/text interaction, instant feedback, and detailed performance analysis.

Users can log in using Google authentication, upload their resume or provide interview details, and take AI-generated mock interviews either by speaking or typing their answers.

---

#  Features

## Authentication
- Google Login Authentication
- Secure user session management

## Credit System
- New users receive **100 free credits**
- Each interview session costs **50 credits**
- Users are redirected to the **Buy Credits** page when credits run out

---

# AI-Powered Mock Interviews

Users can:
- Upload Resume
- Select:
  - Role
  - Experience Level
  - Interview Mode
  -  Technical Interview
  -  HR Interview

The AI generates:
- 2 Easy Questions
- 2 Moderate Questions
- 1 Hard Question

---

# Voice & Text Based Interviews

Users can answer questions using:
-  Voice Input
-  Text Input

This creates a more realistic interview experience and improves communication practice.

---

# AI Feedback System

For every answer:
- AI analyzes the response

- Suggests improvements
- Evaluates confidence and answer quality

# How It Works

1. User logs in with Google
2. User receives 100 free credits
3. User uploads resume or enters:
   - Role
   - Experience
   - Interview Mode
4. AI generates 5 interview questions
5. User answers using voice or text
6. AI provides feedback after every answer
7. Final report is generated with score and feedback
8. 50 credits are deducted after interview completion
9. If credits become 0, user is redirected to Buy Credits page
